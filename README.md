# AutoSimulate

利用多线程实现的一个批量模拟的小工具。

## 使用方法

安装依赖

```shell
pip install -r requirements.txt
```

然后运行

```shell
python main.py
```

>第一次运行会自动生成配置文件

在config.py中配置好全部参数后，再次运行即可。

## 配置填写指南

### REGULAR

这个配置是Alpha表达式

{}代表datafields

例子: `rank({})`
